# Howdy! 
Welcome to a free and open source Sudoku app created in React Native with the help of ChatGPT!

## How to run:
1. Install dependencies and start server
'''bash
yarn install
yarn start
'''
2. Download Expo Go from your phone store
3. Scan the QR Code
